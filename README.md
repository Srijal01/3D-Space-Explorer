# Interactive Particle System

A beautiful, interactive particle system built with HTML5 Canvas and JavaScript.

## Features

- **Real-time particle animation** with 100+ customizable particles
- **Mouse interaction** - particles are attracted to your cursor
- **Dynamic connections** - lines drawn between nearby particles
- **Adjustable parameters**:
  - Particle count (50-500)
  - Connection distance (50-300px)
  - Mouse attraction strength (0-100)
- **Responsive design** - works on any screen size
- **Smooth animations** with 60 FPS performance
- **Pause/Resume** functionality
- **Reset** button to regenerate particles

## How to Run

1. Simply open `index.html` in any modern web browser
2. Move your mouse around to interact with the particles
3. Use the control panel to adjust settings
4. Click "Pause" to freeze the animation
5. Click "Reset" to regenerate all particles

## Technologies Used

- HTML5 Canvas
- Vanilla JavaScript (ES6+)
- CSS3 with gradients and animations

## Customization

You can easily customize the visual appearance by editing:
- **Colors**: Change gradient colors in `style.css` and particle colors in `script.js`
- **Physics**: Adjust velocity, damping, and force calculations in the `Particle` class
- **Effects**: Modify shadow blur, opacity, and line widths for different visual styles

Enjoy exploring the particle system!
